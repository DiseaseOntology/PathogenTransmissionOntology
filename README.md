# PathogenTransmissionOntology

The Pathogen Transmission Ontology (TRANS) describes the transmission methods of human disease pathogens — how a pathogen is transmitted from one host, reservoir, or source to another host. Transmission may occur directly or indirectly and may involve animate vectors or inanimate vehicles.

**Prefix:** TRANS
